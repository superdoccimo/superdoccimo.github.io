📊 Excel統合練習用サンプルファイル生成ツール

Power Query、Python、VBAの練習に最適なサンプルデータを生成します

1️⃣ サンプルExcelファイル生成

各支店の売上データを模したExcelファイルを生成します。すべて同じ構造なので、統合練習に最適です。

2️⃣ Pythonスクリプト

上記で生成したExcelファイルを統合するPythonスクリプトです。

# excel_merge.py - Excel統合スクリプト
import pandas as pd
import glob
import os
from datetime import datetime

def merge_excel_files(folder_path, output_file):
    """指定フォルダ内のExcelファイルを統合"""
    
    start_time = datetime.now()
    print(f"処理開始: {start_time.strftime('%Y-%m-%d %H:%M:%S')}")
    
    # Excelファイルを取得
    excel_files = glob.glob(os.path.join(folder_path, "*.xlsx"))
    print(f"見つかったファイル数: {len(excel_files)}")
    
    df_list = []
    for file in excel_files:
        print(f"読み込み中: {os.path.basename(file)}")
        df = pd.read_excel(file)
        df['ソースファイル'] = os.path.basename(file)
        df_list.append(df)
    
    # 統合して保存
    if df_list:
        merged_df = pd.concat(df_list, ignore_index=True)
        merged_df.to_excel(output_file, index=False)
        print(f"\\n✅ 統合完了！")
        print(f"出力: {output_file}")
        print(f"総行数: {len(merged_df)}")

if __name__ == "__main__":
    merge_excel_files(".", "統合データ.xlsx")

3️⃣ VBAマクロ入りExcelファイル

統合マクロが組み込まれたExcelファイルです。ボタンをクリックするだけで統合できます。

4️⃣ 使い方ガイド

Power Queryで統合する場合

  1. 上記「全支店分を一括生成」ボタンで4つのファイルをダウンロード
  2. すべてのファイルを同じフォルダに保存
  3. 新しいExcelを開き、データ → データの取得 → フォルダーから を選択
  4. 保存したフォルダを指定して統合

Pythonで統合する場合

  1. サンプルExcelファイルとPythonスクリプトを同じフォルダに保存
  2. コマンドプロンプトでそのフォルダに移動
  3. pip install pandas openpyxl を実行（初回のみ）
  4. python excel_merge.py を実行

VBAマクロで統合する場合

  1. 統合マクロ.xlsmをダウンロード
  2. ファイルを開き、「コンテンツの有効化」をクリック
  3. シート上の「統合実行」ボタンをクリック
  4. 統合したいファイルが入っているフォルダを選択
